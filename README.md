# cadence-voice

Communication and presence — prose craft, AI pattern detection, and the skills that shape how you show up professionally. Part of the [cadence](https://github.com/cameronsjo/cadence) ecosystem.

## Skills

| Skill | Source | Use when |
|---|---|---|
| `writing` | Craft + reader-impact framework | Writing or editing professional prose — blog posts, newsletters, technical writing, decline emails, incident reports, white papers |
| `building-personal-brand` | Gartner research methodology | Developing or refining a personal brand strategy — image, impressions, impact |
| `analyzing-competitors` | Manual-invoke only (`disable-model-invocation`) | Analyzing a competitor from a GitHub URL with deep dive and landscape update |
| `presenting-data` | Gartner G00818015 (data storytelling) | Presenting data findings as a narrative or building a data-driven argument |
| `resolving-conflicts` | Gartner G00775161 (political attackers) | Navigating or de-escalating a workplace conflict, especially political attacks |

## Why this plugin

External communication and professional impact form a coherent domain. Writing is "what to say"; presence is "how to show up." Splitting them into separate plugins (the historical cadence-writing and cadence-presence) was an artificial divide — both addressed the same audience and rhythm. Combined here as `cadence-voice` (2026-05-10).

## Install

Available through the [workbench](https://github.com/cameronsjo/workbench) marketplace.

```bash
claude plugin install cadence-voice@workbench
```
